# CMakeHelloWorld

## **How To Build**

1. Crete a build folder:

   ```
   mkdir build
   ```

2. Enter the build folder

   ```
   cd build
   ```

3. Configure with CMake:

   ```
   cmake ..
   ```

4. Buid the project:

  - Debug mode:

     ```
     cmake --build . --cofnig debug
     ```

  - Release mode:

     ```
     cmake --build . --cofnig release
     ```
